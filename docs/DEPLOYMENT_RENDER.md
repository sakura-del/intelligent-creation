## 📋 部署前准备

### ✅ 必须完成的步骤

#### 1. 推送代码到 GitHub

```bash
git init
git add .
git commit -m "feat: 企业级优化完成"
git remote add origin https://github.com/YOUR_USERNAME/AI-Resume.git
git push -u origin main
```

#### 2. 注册 Render 账号

1. 访问 <https://render.com>
2. 点击 "Get Started Free"
3. 选择 "Sign up with GitHub"
4. 授权 Render 访问你的仓库

#### 3. 准备环境变量

在 Render Dashboard 中配置以下环境变量（**必须设置**）：

| 变量名                  | 说明             | 示例值                                               |
| -------------------- | -------------- | ------------------------------------------------- |
| `JWT_SECRET`         | JWT签名密钥        | Render自动生成 ✅                                      |
| `JWT_REFRESH_SECRET` | 刷新令牌密钥         | Render自动生成 ✅                                      |
| `DB_HOST`            | TiDB Cloud主机地址 | `gateway01.ap-southeast-1.prod.aws.tidbcloud.com` |
| `DB_PORT`            | 数据库端口          | `4000`                                            |
| `DB_USER`            | 数据库用户名         | `4Hy5RcbPvLCppKj.root`                            |
| `DB_PASSWORD`        | 数据库密码          | 你的TiDB密码                                          |
| `GLM_API_KEY`        | GLM API密钥 (可选) | 你的API密钥                                           |

## 🚀 一键部署（3种方式）

### 方式A：使用 render.yaml（推荐 ⭐）

**最简单！Render 会自动读取配置并部署。**

#### 步骤：

1. **确保 render.yaml 在仓库根目录** ✅ (已创建)
2. **在 Render 创建 Web Service**
   ```
   Dashboard → New → Web Service
   → Connect your GitHub repo: AI-Resume
   → Render will detect render.yaml automatically!
   ```
3. **点击 "Create Web Service"** - 完成！

Render 会自动：

- 构建前端 + 后端
- 配置环境变量
- 启动服务
- 提供 HTTPS 域名

***

### 方式B：手动配置（更灵活）

#### 步骤1：创建后端服务

```
Dashboard → New → Web Service → Connect GitHub repo
```

**Build Settings:**

| 设置项             | 值                     |
| --------------- | --------------------- |
| Runtime         | Docker                |
| Dockerfile Path | `./server/Dockerfile` |
| Context         | `./server`            |

**Environment Variables:**

```env
NODE_ENV=production
PORT=10000
JWT_SECRET=<auto-generated>
DB_HOST=gateway01.ap-southeast-1.prod.aws.tidbcloud.com
DB_PORT=4000
DB_USER=your-tidb-user
DB_PASSWORD=your-tidb-password
DB_NAME=test
DB_SSL=true
```

**Advanced:**

- Health Check Path: `/health`
- Plan: **Free**

点击 **Deploy** 🚀

#### 步骤2：创建前端服务

```
Dashboard → New → Static Site → Connect same repo
```

**Build Settings:**

| 设置项               | 值               |
| ----------------- | --------------- |
| Build Command     | `npm run build` |
| Publish Directory | `./dist`        |

**Environment Variables:**

```env
VITE_API_BASE_URL=https://YOUR_BACKEND_NAME.onrender.com/api
```

**Routes Configuration:**

```yaml
routes:
  - type: rewrite
    source: /api/*
    destination: https://YOUR_BACKEND_NAME.onrender.com/*
```

点击 **Deploy** 🚀

| 问题              | 原因       | 解决方案                          |

## 🌐 绑定自定义域名（可选）

### 步骤：

1. **Render Dashboard** → 你的服务 → Settings → Custom Domains
2. **Add Custom Domain** → 输入你的域名 (如 `ai-resume.com`)
3. **DNS 配置**（在你的域名服务商）:
   ```
   Type: CNAME
   Name: @ (或 www)
   Value: xxx.onrender.com
   TTL: 300
   ```
4. **等待生效** (通常5-10分钟)
5. **启用 SSL** (Render 自动提供 Let's Encrypt)
